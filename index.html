About me

I'm a Web Developer that seeks to specialize in responsive, multi-platform development using web technologies. While I've done mobile development for android and iOS in the past with Xamarin (C#), I'm currently focusing on front end development with JS/HTML/CSS utilizing modern web-dev frameworks technologies such as AngularJS, SASS, NodeJS, MongoDB, and Ionic (Cordova). All of this, of course while making use of Continuous integration with Gulp/Grunt as well as Scrum AGILE methodologies. My end goal is to become a full stack developer with a strong notion of the entire software development cycle with a focus on web applications that work and adapt in a wide array of environments.
